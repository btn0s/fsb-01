## Product Architecture & Memory System

- Individual memory essential foundation for institutional memory
- Desktop app overlay manages personal context and task assignment
- Task memory most critical differentiator vs existing solutions

**Memory collaboration with human-in-loop:**

- Agent proactively asks for missing context
- Suggests people/documents to consult
- Automated context gathering with user permission

**Source of truth engine concept:**

- Track document ownership and decision history
- Alert when conflicting information emerges
- Generate artifacts (PRDs, code demos) only when needed
