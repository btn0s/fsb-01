# PRD: Onboarding Flow v1 (Deprecated)

**Status:** Deprecated  
**Date:** May 2025  
**Author:** Product Team

## Overview

Multi-step wizard for new user onboarding.

## What We Built

- 5-step wizard flow
- Progressive disclosure of features
- Required profile completion before access

## Results

- **Activation rate:** 32% (target was 60%)
- **Drop-off point:** Step 3 (department selection)
- **User feedback:** "Too many questions before I can use the app"

## Lessons Learned

1. Users want immediate value, not setup
2. Required fields before access kills activation
3. Progressive profiling > upfront questionnaire
4. Mobile users especially frustrated with long forms

## Recommendation

Abandon wizard approach. Consider:

- Immediate access with contextual prompts
- Learn preferences from behavior, not questions
- Optional profile completion over time
