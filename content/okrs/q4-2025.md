# Q4 2025 OKRs - Workplace Platform Team

## Objective 1: Accelerate Feature Velocity

**Key Results:**

- Reduce average feature cycle time from 6 weeks to 2 weeks
- Ship 3 major features per quarter (up from 1)
- Achieve 80% of features launched with positive user feedback

## Objective 2: Improve User Activation

**Key Results:**

- Increase Day-1 activation rate from 45% to 70%
- Reduce onboarding drop-off by 50%
- Achieve 60% feature discovery within first week

## Objective 3: Platform Reliability

**Key Results:**

- Maintain 99.9% uptime for core services
- Reduce P1 incidents by 40%
- Achieve <200ms p95 latency for all API endpoints

## Notes

- Activation is the top priority per leadership
- Previous onboarding wizard experiment failed (May 2025) - users dropped off at step 3
- Focus on low-friction patterns that don't require heavy setup
